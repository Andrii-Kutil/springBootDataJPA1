Spring Data JPA удобный механизм взаимодействия с БД.
Благодаря аннотации @JpaRepository можно не писать примитивные методы как мы это делали в hibernate.
Можно создавать метод по определенным правилам и JPARepository подхватит это.

По умолчанию spring boot использует hibernate. Если мы не хотим его использовать то можем поменять.
Реализация JPA не использует сессию. Сразу пишем Query.

Роль сессии в JPA выполняет EntityManager

spring boot упрощает spring
1)автоматизация конфигурации(к примеру упрощенная схема создания бинов для конекта к БД)
2)управление зависимостями - стартовый пакет
3)встроенная поддержка сервера

@SpringBootApplication вмещает в себе другие аннотации
